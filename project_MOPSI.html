Usability testing for MOPSI

MOPSI is a location based information system developed by SIPU lab, Computer science department, University of Eastern Finland

Analysis of Context of Use

First I analyzed the context of use of Mopsi in dimension of user, task and envrionment

User tasks

I defined several user tasks for testing.

User profile of the testing

6 users took the testing.

  • 3 international student and 3 locals
  • none of them has experience in using the system before

Test setting

Tests were conduct at home of moderator(Me) or home of users.

Users were allowed to explore the system before testing.

No training was given to users.
