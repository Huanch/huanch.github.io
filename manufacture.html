Youjia Saas for Automative Manufacture





Company: Baidu

Time:2021.08

Role: Rquirement analysis, UI design

Youjia is a car selling platform for buyers and retailors

Youjia is an online car information platform, which provides information for consumers and marketing solution for car retailors.



We need to provide an application for manufactures.

We hope the manufacturers would think our platform is effective and purchase our solutions for their retailers. So we want to offer an application for the manufacturers to monitor how many customer their retailers get from our platform.

Defining the problems and explore solutions.

For each function unit, i analyzed the goals and the problems we need to solve by designing interface and define possible solutions.



How can user find target vehicle efficiently when setting the limit price?

While designing the table of vehicle types for limit price management, we need to help user to find the target vehicle easily and efficient.

I defined 2 alternatives solution for the table.



The solution

Retailor list

User sometime needs to check if the retailor data are correct. We give user a horizontal bar for location filters, so user can quickly switch data by location and check the correctness of retailor data location by location.



Retailor profile

A car brand usually has a large amount of car models. When editing or verifying the list of car models with selling permission in retailor profile, user might have difficulties in find the models from a long list and check the accuracy of setting. I decided to use two listboxes with search fuction, one for unchecked models and one for checked models.

Customer contact

user can browse contacts retailers obtain from Youjia platform.

As users need to check the data by locations and like to pay special attention on data in current week and month, I designed a filter area.

Vehicle management

user can manage the price restriction of each vehicle types for sell on Youjia platform.

Vehicle types under a car model are listed on individual tab. Users can find a vehicle typers by selecting the car model tab.

Creating the design handover

I defined UI specification to help developer to implement the design with front end code.
